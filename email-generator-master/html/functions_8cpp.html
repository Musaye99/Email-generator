◆ generateEmail()

QStringList generateEmail ( const QStringList & temp,
const QStringList & data )

Generates an email from a template and data.

Replaces placeholders in the template (e.g., #name#) with values from the data list. The data should be in the format "key: value", where "key" matches the placeholders in the template.

Parameters
tempA list of lines from the email template.
dataA list of key-value pairs to replace placeholders in the template.
Returns
QStringList The generated email with placeholders replaced.
